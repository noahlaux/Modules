Class: fb

LBI.fb

new fb

Bridge to the new Facebook SDK with oauth 2, that loads the SDK asyncronously, creates the FaceBook DOM root element and defaults to auto grow the application iframe

Author:
  • Noah Laux (noah.laux@lbi.com)
Source:
  • /Applications/MAMP/htdocs/Modules/assets/js/LBI/LBI.fb.js, line 1
Examples
// Initiate with apikey (mandatory)
LBI.fb.init({ apiKey: '281411815254963' })
// Initiate and check if user likes a page
LBI.fb.init({apiKey: '281411815254963'}, function(fb) {
      if ( fb.hasLiked('125289967525962') ) {
          // User likes the page
      } else {
          // User do not like the page
      }
});
